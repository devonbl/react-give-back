# react-give-back

An modal that pops up with an iframe in it to donate to a [**Tilt campaign for Hack the Hood**](https://www.tilt.com/tilts/hack-the-hood) embeddable within a react component or whatever

## use it
Be like
````
<ReactGiveBack />
````
